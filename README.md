This is a P2P file transfer system that will allow 
the sending, recieving and detection of data chunks 

You will be able to:
- Load a config 
- load package files and parse their format 
- manage files that are completed and incomplete 
- Complies with a network protocol to communicate with peers 
- Inform a client program of the latest info of your peers and the files the program manages
- Finalise and check that downloaded files match expected file 
- able to shutdown and disconnect from peers
